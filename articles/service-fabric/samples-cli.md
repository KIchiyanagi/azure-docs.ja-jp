次の表に、Service Fabric クラスター、アプリケーション、およびサービスを管理する [Service Fabric CLI](service-fabric-cli.md) のサンプル スクリプトへのリンクを示します。

[!INCLUDE [links to azure cli and service fabric cli](../../includes/service-fabric-sfctl.md)]

| | |
|-|-|
