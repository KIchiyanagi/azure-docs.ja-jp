# <a name="azure-ad-access-reviews-preview"></a>Azure AD アクセス レビュー (プレビュー)

組織でアクセス レビューを使うことにより、グループ メンバーシップおよびエンタープライズ アプリケーションへのアクセスを、効率的に管理できます。 

## <a name="what-can-you-do-with-access-reviews"></a>アクセス レビューでできること
   
- アプリケーションへのアクセスおよびグループのメンバーシップのアクセス レビューを使うと、ゲスト ユーザーが引き続きアクセスする必要があるかどうかをレビュー担当者が効率的に決定できる洞察により、ゲスト ユーザーのアクセスを認定することができます。
      
- アクセス レビューを使って従業員のアプリケーション アクセスとグループ メンバーシップを認定できます。
   
- アクセス レビューの制御を組織に関連するプログラムに集めて、コンプライアンスまたはリスクを受けやすいアプリケーションのレビューを追跡できます。
